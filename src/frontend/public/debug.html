Debug Page

Check the browser console for errors.

If you see this page, the server is running correctly.

Navigate to http://localhost:3000 to see the app.

Debugging Steps:

  1. Open browser developer tools (F12)
  2. Check the Console tab for any red errors
  3. Check the Network tab to see if all files are loading
  4. Common issues:
    • Missing dependencies
    • Import errors
    • Syntax errors in components
    • Wallet adapter configuration issues
